# pgEdge MCP Bridge

The pgEdge MCP Bridge is an HTTP-to-stdio proxy for the Model Context Protocol (MCP). It enables MCP clients that only support HTTP/SSE transport to communicate with MCP servers that use stdio transport.

## Features

- HTTP/SSE to stdio protocol bridging
- OAuth 2.0 authentication support
- TLS encryption
- CORS configuration
- Session management
- Flexible configuration via YAML
